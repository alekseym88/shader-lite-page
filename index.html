Shader Node Preview Lite is a simplified, free version of Shader View. It gives you a real-time preview of the currently selected shader node — so you can instantly see what a node is doing without constantly rewiring your material.

It uses the same way as Shader View, but with one key limitation: only one node can be previewed at a time. Perfect if you want to try the workflow, learn shaders, or keep things super lightweight.

FAQ

What Blender versions are supported?

Shader Node Preview Lite uses the same core engine as Shader View and is tested on Blender 3.0 through 5.x on macOS, Windows, and Linux.

Does it modify my node tree?

No. The add-on is completely non-destructive. It evaluates your node tree and draws previews on top of nodes, but never inserts, deletes or edits any nodes, links, or scene data.

Will Lite slow down Blender?

Lite is very lightweight: it renders previews for one selected node. For heavy scenes you can always temporarily disable the add-on, or switch to the full Shader View, which includes advanced performance controls and background processing.
